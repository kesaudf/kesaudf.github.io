Applications

Revit

  • Copy elements from files into current file.
    This app allows you select any file, including link files. Application will able to transfer following element types - ViewTemplates, KeySchedule, RebarBarType, LinePattern, ElectricalLoadClassification ,Materials, ExportDWGSettings, ElectricalSettings, ElectricalDemandFactorDefinition, BrowserOrganisation, ReinforcementSettings, DuctSettings, CableTraySettings, ConduitSettings
    Video
  • Import strings into key schedules.
    This app allows you insert selecting strings into keyschedules from another key schedules from selected files. Selected and pasted specification names have to be match.
    Video
  • Export sheets in PDF-format.
    This app allows you exporting sheets into PDF-format. Program defines all standard and non-standard sheet formats. Program applies PDF Creator printer in its process.An export operation can be performed with a current file or external link files.
    Video

  • Create a room schedules by sections and levels.
    This app allows you to create a room schedules where filters divided by model sections or/and levels. Also this program make a combined filters by selecting levels and sections.
    Video
  • Update and copy families from selected files.
    Users select a required families in a current file for copy or upgrade in another files. Also user selects needful file which open in Revit window on a program form. Application exports a necessary families and inserts or updates them in selected files.
    Video
  • Detach Central files with discard worksets.
    User selects a necessary Central files and these files will detach and save to selected or default folder
    Video
  • Fill a reinforcement parameter values by values from base elements.
    User selects a necessary family categories and application fills reinforcement values by values from base elements of selected categories.
    Video
